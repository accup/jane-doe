# Jane Doe

AIとの対話における長期記憶と文脈の継続性を実現するシステム

## 解決する課題

AIは現状として、時間的・空間的な隔たりのある情報を記憶によって結びつけることが難しい。このプロジェクトは、その制約を克服し、継続的な文脈を維持した対話を可能にする。

## アーキテクチャ

```
Slack App (対話の環境)
    ↓
Claude API (記憶の分配、管理、および対話文の構築)
    ↓
MCP (記憶の検索、更新)
```

### 構成要素

- **Slack App**: ユーザーとの対話インターフェース
- **Claude API**: 記憶の整理・分配・管理、文脈を考慮した応答の生成
- **MCP (Model Context Protocol)**: 長期記憶の保存、検索、更新

## 開発方針

1. 自分用の最小構成から開始
2. 動作を検証し、確信を持って進める
3. 段階的にスケールさせる
